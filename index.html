Connection instructions:

Pidgin

  • Open Pidgin

  • Press "Accounts"

  • Go to "Manage Accounts"

  • Press "Add..."

  • Set the protocol to "XMPP"

  • Enter a username

  • Set the domain to "chat.rubberroomwithrats.com"

  • Set a password

  • Check "Create a new account on the server

  • Press "Add"

Conversations

  • Open Conversations

  • Press "Manage Accounts"

  • Press the user+ icon

  • Check "Register new account on the server"

  • Set the XMPP address to "YOURUSERHERE@chat.rubberroomwithrats.com"

  • Set a password

  • Press "Next"
